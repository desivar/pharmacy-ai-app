💊 AI-Powered Pharmacy Management System
A modern, intelligent dashboard for pharmacies to manage stock, predict sales, and communicate with suppliers using AI.

🌟 Key Features
Role-Based Access: Secure login for Admin (User management) and Staff (Inventory tasks).

Smart Inventory: Search, Sell, Add, and Edit medicines with a TypeScript/React interface.

AI Alerts: Automated WhatsApp supplier messages generated by Groq Cloud AI when stock is low or expiring.

Sales Predictions: AI-driven forecast cards to stay ahead of demand.

Internal AI Assistant: A dedicated chat window to ask questions like "Which medicine will run out first?"

User Management: Admins can create and manage staff accounts.

🚀 Tech Stack
Frontend: React 18, TypeScript, Vite, Tailwind CSS / Lucide Icons.

Backend: FastAPI (Python 3.11+), SQLite Database.

AI Engine: Groq Cloud API (Llama 3/Mixtral) — Zero local cost, lightning-fast inference.

Communication: WhatsApp API integration for supplier alerts.
